What is Styled Components?

  • Created by Max Stoiber & Glen Maddern
  • Create React components for styling with plain CSS
  • Tagged Template Literals 
    someFunction`template string ${me} magic`
    someFunction(['template string ', ' magic'], me)
  • CSS Contents get hashed and injected into head

Why Styled Components?

  • Write plain, old CSS
  • Covers all CSS
  • Separation of Concerns & Colocation
  • Dynamic Styling & Extensible

It's just CSS!


import styled from 'styled-components'

const MyButton = styled.button`
  color: red;
  outline: none;
  background: white;
  border: 1px solid red;
  border-radius: 5px;
`

<MyButton onClick={() => console.log('clicked')}>Click Me</MyButton>

Covers All CSS


const FlexContainer = styled.div`
  display: flex;
  @media (min-width: 768px) {
    flex-direction: column;
  }
`

const HoverButton = styled.button`
  background: white;
  color: red;
  border: 1px solid red;
  &:hover {
    background: red;
    color: white;
  }
`

Separation of Concerns

Colocate Styles with Components


const FlexContainer = styled.div`...`
const HoverButton = styled.button`...`

const MyComponent = ({thingOne, thingTwo, handleClick}) => {
  return (
    <FlexContainer>
      <SomeComponent someProp={thingOne} />
      <OtherComponent thisProp={thingTwo} />
      <HoverButton onClick={handleClick}>Suessify</HoverButton>
    </FlexContainer>
  )
}

Dynamic Styling with Props


const DynamicButton = styled.button`
  color: ${props => props.color || 'red'};
`

<DynamicButton /> //red
<DynamicButton color="cornsilk" /> //kinda yellow brown?

Extend Existing Components


//React-Router Link Component
const PrettyLink = styled(Link)`
  color: papayawhip;
  text-decoration: none;
`

//Extend an existing styled component
const BlackHoverButton = HoverButton.extend`
  color: black;
`
